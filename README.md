# Shohay


## Installation

Use Git bash and [node package manager](https://www.npmjs.com/) or [yarn package manager](https://yarnpkg.com/en/) to install the project

```bash
git clone https://github.com/nayan2/Shohay.git
cd Shohay
react-native eject
npm install
react-native link
```
